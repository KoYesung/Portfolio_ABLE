<h1>Project 개요</h1>

<h3> 💡아이디어 </h3>

* 서울시 열린데이터광장 공공데이터를 활용하여 교통약자를 위한 편의시설을 알려주는 통합 지도앱
* 길찾기, 지하철, 장애인 콜택시, 의료시설 4가지 메인 메뉴로 구성
* 회원가입, 로그인, 즐겨찾기, 주소관리 등 부가 기능 구현
<br>
<h3> 🏆수상내역</h3>

* 서울시 열린데이터 광장 경진대회 우수상 수상
<br>  
<h3> 👩‍💻팀 인원 및 역할 </h3>

* 전반적인 앱 디자인, 로고 제작
* 지하철 편의시설 안내 페이지
* 데이터베이스 구축
* 즐겨찾기, 주소관리 서버 라우팅
<br>
